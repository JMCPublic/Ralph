## 28 April 2026

**What we built**
- Ralph live on the internet at https://jmcpublic.github.io/Ralph/
- Mobile-first design with voice search (tap mic, say a brand name, get a result)
- Desktop QR code nudge — non-mobile visitors see a full screen prompt with a QR code pointing to the mobile site
- Smart not-found card — searching an unknown brand (e.g. TSB) now returns "No TSB link yet — be the first" with a pre-filled mailto button for that brand
- Search chips — Monzo, Freetrade, Chase, Wise, Octopus Energy, TSB
- Mic animation — sonar rings pulse outward when voice search is listening

**Design decisions**
- Palette: navy & cream (Option C — #185FA5, #042C53, #FDFCF8)
- Current font: EB Garamond headings, DM Sans body
- Font to explore later: Outfit (clean geometric, modern fintech feel)

**Product thinking**
- Identified ShareMyLink.co.uk as a comparable — concluded Ralph's differentiator is the fairness narrative and the warm entry point, not just the mechanic
- Ruled out WhatsApp Business API — too expensive, Meta as gatekeeper is a risk
- Telegram ruled out — reputation problem
- Solution: mobile-first web with voice search replicates the WhatsApp bot experience without any API dependency

**Ideas parked (loyalty layer)**
- Piggyback message: when Michael submits a link Ralph replies asking him to forward a ready-made Ralph introduction to his contacts
- Draw mechanic: entries earned per contact who signs up via Michael's message
- Retroactive reward: when Ralph can afford prizes, go back to day-one sharers and include them
- Prize choice: curated list or donate to charity
- Charity option: Ralph randomly picks 3 smaller lesser-known charities, Michael chooses one

**Tools and skills learned**
- Created a GitHub account from scratch
- Created a public repository
- Navigated file naming issues on Windows (hidden extensions, .html.html)
- Uploaded and replaced files on GitHub Pages
- Published a live website via GitHub Pages
- Previewed HTML files locally before publishing
